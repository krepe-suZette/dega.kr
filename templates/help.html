기본 이용 방법

클랜 목록 페이지에서 내 클랜을 찾아줍니다. 내 클랜이 목록에 없다면 클랜 추가 신청을 해주세요.

클랜 페이지로 들어가면 약간의 로딩 후 온라인 멤버 목록을 볼 수 있습니다.

  1. 클립보드로 복사
    누르면 "/합류 <합류코드>"가 클립보드로 복사됩니다. 게임 내 채팅창에서 Ctrl + v를 통해 붙혀넣을 수 있습니다.
  2. 새로고침
    현재 온라인 멤버 목록으로 새로고침합니다. API상의 문제로 실시간 반영이 되지 않을 수 있습니다.
  3. 이전/다음 페이지
    이전 페이지 또는 다음 페이지로 이동합니다.
  4. 클랜 정보 바로가기
    새 창에서 Bungie.net의 해당 클랜 소개 페이지를 엽니다.

클랜 추가 신청

추가 예정입니다.

스팀 오버레이 웹 브라우저에서 사용하기

dcde.ga를 스팀 오버레이 웹 브라우저와 함께 사용하면 더욱 간편하게 서비스를 이용할 수 있습니다.

게임 화면에서 스팀 오버레이 단축키(기본값: Shift+Tab)을 사용하면 위와 같은 화면이 나옵니다.

  1. 웹 브라우저
    실행시 설정한 홈페이지가 오버레이 안에서 열립니다.
  2. 설정
    웹 브라우저를 비롯한 설정이 가능합니다.

설정 > 웹 브라우저를 눌러주세요. 웹 브라우저 홈페이지(3)를 자신이 속한 클랜 페이지 주소로 변경하면 더 편하게 이용이 가능합니다.

주의사항

Bungie.net 설정의 "계정 & 연결" 탭에서 "공개 프로필에 표시" 설정을 활성화하지 않은 경우 온라인이더라도 합류 코드 복사가 불가능 할 수 있습니다. 설정을 다시 해준 후 관리자에게 요청해주세요.
