합류 코드가 무엇인가요?

더이상 사용되지 않습니다.

데스티니 가디언즈에서 다른 수호자의 화력팀에 합류하기 위하여 사용되었던 17자리의 숫자입니다. 스팀에서 유저 식별을 위해 사용하는 steamID64와 동일하며, 자신의 스팀 ID는 스팀 프로필의 기본 주소를 확인하거나 게임 채팅창에 /아이디를 입력하면 알 수 있습니다.

게임 채팅창에 /합류 <합류코드>를 입력하면 해당 수호자의 화력팀에 합류를 시도하게 되고, /초대 <합류코드>를 입력하면 해당 수호자에게 자신이 속한 화력팀의 초대 메시지가 전송됩니다.

번지 이름은 무엇인가요?

합류 코드를 대체하여 데스티니 가디언즈 3.3.0 업데이트 (잃어버린 자 시즌, 2021-08-25) 이후부터 사용하게 되는 새로운 이름입니다. 닉네임#0000 형태로 되어있으며, 뒤의 4자리 숫자는 무작위로 정해집니다. 게임 내에서 F1 버튼을 눌러 확인할 수 있습니다.

게임 채팅창에 /합류 <번지 이름>을 입력하면 해당 수호자의 화력팀에 합류를 시도하게 되고, /초대 <번지 이름>을 입력하면 해당 수호자에게 자신이 속한 화력팀의 초대 메시지가 전송됩니다.

기본 이용 방법

dega.kr은 데스티니 가디언즈의 플랫폼 이전으로 인해 생겨난 클랜 수호자 명단 표시 오류로 인한 불편함에서 시작되었습니다. 친구 추가를 하지 않더라도 상대의 화력팀 설정이 공개되어있다면 바로 합류할 수 있도록 대체 클랜 수호자 명단을 만드는 것을 목표로 하였습니다.

클랜 목록 페이지로 이동한 다음 자신의 클랜을 찾습니다. 만약 내가 속해있는 클랜이 목록에 없다면 클랜 추가 신청을 해주세요.
깃발을 클릭하면 해당 클랜을 목록의 최상단에 고정할 수 있습니다.

클랜 페이지로 들어가면 현재 게임에 접속중인 수호자 명단을 볼 수 있습니다. 각 버튼들의 역할은 아래와 같습니다.

  • : 해당 수호자에게 바로 합류를 시도합니다. 브라우저에서 실행하는 경우 알림이 뜰 수 있습니다.
  • : /합류 <번지 이름> 명령어가 클립보드로 복사됩니다. Ctrl 버튼을 누른 상태로 버튼을 누르면 /초대 명령어가 대신 복사됩니다.
  • 접속중인 수호자 목록을 새로고침합니다. 키보드 R을 눌러도 사용할 수 있습니다.
  • 이전 페이지 또는 다음 페이지로 이동합니다. 키보드 A와 D, 왼쪽 방향키와 오른쪽 방향키도 사용이 가능합니다.
  • 새 창에서 Bungie.net의 해당 클랜 소개 페이지를 엽니다.
  • 설정 페이지로 이동합니다.

클랜 추가 신청

자신의 클랜 이름을 정확히 입력해 검색해주세요. 이름을 올바르게 입력했다면 클랜의 정보가 출력됩니다. 오른쪽 아래의 "등록 신청" 버튼을 눌러주세요.
성공적으로 등록 신청이 이루어진 경우 일정 시간 이후 자동으로 클랜 목록에 반영됩니다. 일반적으로 최소 (클랜원 수 * 2.5초)의 시간이 필요합니다.

클랜 등록 신청 조건을 충족하지 못하는 경우 등록이 불가능합니다. 현재 클랜 등록이 불가능한 경우는 아래와 같습니다.

  • 클랜 구성원이 10명 이하일 때
  • 클랜 관리자의 요청이 있었을 때

스팀 오버레이 웹 브라우저 활용

자신의 dega.kr 클랜 페이지를 스팀 오버레이 웹 브라우저의 홈 페이지로 설정하면 더욱 완벽하게 서비스를 이용할 수 있습니다.

게임 화면에서 스팀 오버레이 단축키(기본값: Shift+Tab)을 사용하면 위와 같은 화면이 나옵니다.

  1. 웹 브라우저
    실행시 설정한 홈페이지가 오버레이 안에서 열립니다.
  2. 설정
    웹 브라우저를 비롯한 설정이 가능합니다.

설정 > 웹 브라우저를 눌러주세요. 웹 브라우저 홈페이지(3)를 자신이 속한 클랜 페이지 주소로 하게 되면 (1) 웹 브라우저를 눌렀을 때 바로 클랜원 목록이 뜨게 됩니다.

설정

dega.kr의 설정들이 모여있는 페이지입니다. 설정 값은 기기에 저장됩니다.

  • 모드 설정
     바로 합류 버튼을 띄울지, 복사 버튼을 띄울지 선택합니다. 기본값은 Windows 환경에서만 바로 합류, 나머지 환경에서는 복사 입니다. 현재 사용되지 않습니다.
  • 명령어 복사 언어 설정
     복사 명령어의 언어를 선택합니다. 한국어 클라이언트를 사용하시지 않는다면 영어를 선택하시면 됩니다. 기본값은 한국어입니다.
  • 내 정보 등록
     기재한 번지 이름과 일치하는 사람이 클랜원 목록에 있는 경우 밑줄이 생깁니다. 자신의 번지 이름을 적어두시는 것을 권장합니다.
  • 창 활성화시 업데이트
     브라우저의 탭/창/오버레이 등을 최소화했다가 다시 표시했을 때 자동으로 새로고침을 시도할지에 대한 설정입니다. 기본값은 활성화입니다.
  • 돌아가기
     이전 페이지로 돌아갑니다. 브라우저의 뒤로 가기 버튼과 동일한 역할입니다.
